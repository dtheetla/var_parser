# VarParser


VarParser extracts the instance variables in a Ruby code.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'var_parser'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install var_parser

## Usage
```ruby
require "var_parser"
VarParser::extract_instance_vars(file path string)
```
